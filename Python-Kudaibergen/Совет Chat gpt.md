## 1. Учить не функции, а **паттерны**

Вместо:

“надо запомнить `join`, `map`, `filter`…”

Думай так:

- работа со строками → `split`, `join`
- перебор → `for`, `enumerate`
- трансформация → list comprehension
- фильтрация → `if` в списках / `filter`

👉 Мозг лучше запоминает **сценарии**, а не названия

---

## 🔁 2. Повторение через практику (самое эффективное)

Схема:

1. Решил задачу (как ты сделал с реверсом)
2. Потом смотришь “как делают по-питоновски”
3. Переписываешь

Пример:

# сначала
result = []  
for x in a:  
    result.append(x * 2)  
  
# потом
result = [x * 2 for x in a]

👉 Так формируется “чутьё языка” — это уровень Middle

## 📚 3. Делай свой мини-“справочник”

Очень мощная техника:

# строки  
s[::-1]        # реверс  
s.split()      # разбить  
" ".join(lst)  # собрать  
  
# списки  
[x for x in a if x > 0]  
enumerate(a)  
  
# полезное  
zip(a, b)  
sorted(a)

👉 Не зубри — **держи под рукой и регулярно используй**

## 🧩 4. Решай задачи с ограничениями

Например:

- решить **без цикла**
- решить **в одну строку**
- решить **через встроенные функции**

Пример:

# обычный
for i in range(len(a)):  
    print(i, a[i])  
  
# лучше
for i, v in enumerate(a):  
    print(i, v)
